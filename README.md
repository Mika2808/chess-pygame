# Chess Pygame

A chess game built in Python using Pygame, featuring a custom chess engine with bitboard-based move generation and a simple AI opponent.

---

## Features

- Full chess rules implementation
- Bitboard-based move generation (fast and scalable)
- Legal move validation (check detection included)
- Basic AI opponent (minimax + alpha-beta pruning)
- Pygame-based graphical interface
- Click-to-move gameplay

---

## AI

- Currently uses a minimax algorithm with alpha-beta pruning
- Evaluation based on material + piece-square tables
- Black is controlled by AI, white is player-controlled

---

## Tech Stack

- Python 3.x
- Pygame
